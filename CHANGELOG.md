## [1.0.0] - 2024-02-08

### 新增
- 🎉 首次正式发布 PassPro 密码管理器
- 🔐 主密码保护机制（PBKDF2 + SHA256）
- 🛡️ AES-256-GCM 端到端加密
- 🔑 智能密码生成器（自定义长度和字符集）
- 📂 密码分类管理（社交、工作、金融等）
- 🔍 实时搜索功能
- 📋 一键复制用户名/密码
- 💻 Electron 桌面应用（1200x800 窗口）
- 🌐 浏览器模式支持（localStorage 存储）
- 🎨 现代化 UI（React + Tailwind CSS）

### 技术特性
- Electron 28.3.3 + Vite + React 技术栈
- IPC 安全通信机制
- 独立加密工具模块
- 双模式兼容设计

### 安全
- 所有密码数据本地加密存储
- 内存中不保存明文密码
- 主密码哈希存储（非明文）
