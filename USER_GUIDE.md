# PassPro 使用指南

## 🚀 启动应用

### 开发模式
```bash
cd passpro-project
npm run electron:dev
```

### 生产模式（构建后）
```bash
npm run electron:build:win  # Windows
npm run electron:build:mac  # macOS
npm run electron:build:linux # Linux
```

---

## 📱 使用流程

### 1. 首次使用

1. 启动应用，看到 **"欢迎使用"**
2. 设置主密码（至少6位）
3. 点击 **"设置并进入"**
4. 进入主界面

### 2. 日常使用

1. 启动应用，看到 **"欢迎回来"**
2. 输入主密码
3. 点击 **"解锁"**
4. 查看/管理密码

### 3. 添加密码

1. 点击 **"新建项目"** 按钮
2. 填写信息：
   - 服务名称（如：Twitter）
   - 用户名/邮箱
   - 密码（可自动生成）
   - 分类（社交/金融/工作/娱乐）
3. 点击 **"保存项目"**

### 4. 查看密码

- 点击卡片上的 **眼睛图标** 显示/隐藏密码
- 点击 **复制图标** 复制密码到剪贴板
- 点击 **编辑图标** 修改密码

### 5. 分类筛选

- 左侧导航栏选择分类
- 或点击 **"安全概览"** 查看全部

### 6. 搜索密码

- 在顶部搜索框输入关键词
- 实时搜索标题和用户名

### 7. 退出登录

- 点击左侧 **"退出登录"**
- 或关闭应用窗口

## 🔐 安全提示

### 主密码
- 主密码用于加密所有数据
- 请务必牢记，无法找回
- 建议设置强密码（8位以上，包含大小写和数字）

### 数据备份
- 数据保存在：`C:\Users\<用户名>\.passpro\`
- 定期备份 `passwords.enc` 文件
- 可以复制到U盘或云盘

### 自动保存
- 添加/修改/删除密码后自动保存
- 无需手动点击保存按钮

## 🐛 常见问题

### Q1: 忘记主密码怎么办？
**A**: 无法找回，数据永久丢失。请务必牢记主密码！

### Q2: 如何备份密码？
**A**: 复制文件 `C:\Users\<用户名>\.passpro\passwords.enc` 到安全位置。

### Q3: 可以在多台电脑使用吗？
**A**: 可以，复制 `.passpro` 文件夹到新电脑即可。

### Q4: 卸载应用会丢失数据吗？
**A**: 不会，数据在独立的用户目录中。

### Q5: 如何彻底删除所有数据？
**A**: 删除文件夹 `C:\Users\<用户名>\.passpro\`
