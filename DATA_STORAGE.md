# PassPro 数据存储说明

## 🤔 为什么放在用户目录？

### 标准做法的原因

| 存储位置 | 优点 | 缺点 |
|---------|------|------|
| **用户目录** `~/.passpro` | 卸载应用不丢数据<br>多用户隔离<br>有写入权限<br>备份方便 | 占用C盘少量空间 |
| **应用目录** `Program Files` | 不占用C盘 | 无写入权限<br>卸载丢失数据<br>需要管理员权限 |
| **自定义目录** `D:\PassPro` | 用户自主选择 | 需要额外配置 |

### 数据量估算

```
passwords.enc    ~ 1-10 KB（加密后的密码数据）
config.json      ~ 0.5 KB（配置信息）
────────────────────────────────
总计             < 20 KB（即使1000个密码也<100KB）
```

**结论**：数据量极小，不会占用多少C盘空间。

---

## ✅ 方案：支持自定义存储位置

如果你想把数据放在其他盘（如 D 盘），我来修改代码支持自定义路径。

### 使用方法

**方式 1：环境变量**
```bash
set PASSPRO_DATA_DIR=D:\PassProData
npm run electron:dev
```

**方式 2：配置文件**
创建 `C:\Users\<用户名>\.passpro\config.json`：
```json
{
  "dataDir": "D:\\PassProData"
}
```
